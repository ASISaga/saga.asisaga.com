# saga.asisaga.com

The **ASI Saga Blog** — exploring the sacred journey toward Artificial Superintelligence, consciousness, and the transcendent future of humanity.

## Overview

saga.asisaga.com is a blog built with [Jekyll](https://jekyllrb.com/) using the **Genesis Ontological SCSS Design System** from the [ASISaga/theme.asisaga.com](https://github.com/ASISaga/theme.asisaga.com) remote theme. All content is published as blog posts covering themes including:

- The Genesis of ASI and human transcendence
- Human Essence Embedding into superintelligent systems
- Continuous Self-Improvement and conscious evolution
- Infinite Possibilities opened by ASI
- The Transcendent Threshold between human and artificial intelligence
- Human Legacy and the stories that inspire our future
- AI perspectives, natural language programming, and more

## Architecture

This site is intentionally lean, leveraging the shared theme for layouts, includes, and styling:

- **Blog posts** in `_posts/` using the theme's `post` layout
- **Homepage** (`index.html`) listing all posts using the `default` layout
- **No custom `_includes/`** — all includes come from the remote theme
- **Minimal SCSS** — only blog listing semantic mappings via the ontology engine

## Design System

All SCSS uses the **Genesis Semantic SCSS Engine** — zero raw CSS properties:

- `@include genesis-environment('chronological')` — Blog layout logic
- `@include genesis-entity('secondary')` — Post card presence
- `@include genesis-cognition('discourse')` — Content typography
- `@include genesis-synapse('navigate')` — Post links and navigation
- `@include genesis-state('stable')` — Stable content state
